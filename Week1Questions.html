Week #1 Questions and Answers

HTML Question

What is the difference between the class attribute and the id attribute?

The id attribute is unique to a page so only one element can have a particular id.
Many elements can share the same class attribute however.

CSS Question

What is the difference between external stylesheet, internal stylesheet, and inline styling?

Inline styling is using style tags in individual elements in the body.
Internal stylesheet is having a style tag in the head section that contains the css rules.
External stylesheet is using an external file for the css rules, the link to the file is in the head section.
Inline styling overrides internal and external stylesheets.
Internal stylesheet overrides external stylesheet if it comes after the link to the external stylesheet.

Website Structure Question 1

Why is it better to use more descriptive elements like header and section instead of a bunch of divs or spans?

It is better for organization and helping you keep track of what things should go where, it is also useful
for other people who may also be working on the page. Using the div or span by itself doesn't give any
information about what kind of content should be inside.

Website Structure Question 2

Give 3 elements that should be in the head section, and 3 that should be in the body section.

The head section should contain meta data, so things like the doctype, the charset, and links to external css
files (and/or the internal stylesheet) or script files. The body contains the actual elements that will be
displayed on the page, so things like headers, main, sections, articles, and images will be here.
